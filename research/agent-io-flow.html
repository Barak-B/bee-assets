⚙️ 4. צינור טיפול — מה קורה בכל סוג

לכל intent יש sub-pipeline. השלבים: enrich (איסוף קונטקסט) → decide (האם צריך אישור) → compose (טמפלייט) → execute (פעולה אמיתית).

🏠 client-status intent · medium urgency
"מה קורה במערכת של פריים אנרג'י באתר אשקלון?" / "ייצור היום?"
DM/self-chat → router classify → enrich: BEE + SolarEdge → solar-agent compose → self-chat ⚡ *...*
enrich מצריך: alfred-bee.js sites --client=X · alfred-solaredge.js production --site=Y · alfred-bee.js alerts --siteId=Z.
compose משתמש ב-template "client-status" עם 4 שדות: kW נוכחי · kWh היום · השוואה ל-7d · alerts פתוחים.
אם anomaly (drop ≥30% מהממוצע) → escalate ל-client-fault.
Outputs: WhatsApp self-chat ⚡ *...* · log ב-memory/<date>.md
🚨 client-fault / solar-alert intent · HIGH urgency
SolarEdge alarm · "המערכת לא עובדת" · production drop >30%
API poll / DM → classify severity → solar-agent diagnose → propose action → self-chat + maybe Drafts group
Severity buckets: critical (production=0 בשעות שיא) · warning (10-30% drop) · info (noisy alert e.g. NO_PRODUCTION_TODAY בלילה).
Output template:
⚡ *🚨 לקוח X · אתר Y · INVERTER_OFFLINE לפני 2h
ייצור: 0 kW (השוואה: 6.2 kW לפני יום)
פעולה מומלצת: reboot remote / ביקור באתר
לעדכן את הלקוח? [כן/לא]*
אם ברק מאשר → draft ל-Drafts group ללקוח.
אם נדרש ביקור → propose calendar event ב-tasks-shlomi-solar.
Outputs: WhatsApp self-chat · BEE.Alert.acknowledged=true · calendar event proposed · אופציונלי draft ללקוח
💰 client-payment intent · HIGH urgency · sensitive
"שילמתי 5000" · POS event · Gmail "Cardcom payment received"
DM / Gmail / Invoice Maven → BEE.Invoice lookup → amount + match → propose create receipt → Invoice Maven --yes
enrich: alfred-bee.js invoices --clientId=X --status=open. אם יש invoice פתוח עם amount קרוב → match.
propose ב-self-chat: ⚡ *💰 קיבלתי תשלום ₪X מלקוח Y. ליצור קבלה? חשבונית פתוחה #1234 = ₪Y. [כן/לא]*
אחרי "כן" → alfred-invoice-maven.js create-receipt --invoiceId=1234 --amount=X --yes.
BEE.Payment record + Monday board "תשלומים" update.
Outputs: Invoice Maven receipt · BEE.Payment write · Monday update · self-chat confirmation
📋 client-quote intent · medium urgency
