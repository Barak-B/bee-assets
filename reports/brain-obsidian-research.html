3 · חיבור המוח לאובסידיאן (מה נבנה עכשיו)

1. מקור אמת בגיט — כל research/**/*.md (כולל BRAIN) נמשך לענף הזה.
2. ערכת vault ניידת — obsidian-vault/ עם .obsidian/ + 00-START-HERE.md בעברית.
3. סנכרון חוצה־פלטפורמות — PowerShell קיים + Bash חדש: sync-vault-and-graphify.sh.
4. Hook אחרי commit — מעדיף PowerShell, נופל ל־Bash אם אין PS.
5. בדיקה בענן — mirror ל־/tmp/bee-obsidian-mirror: 86 קבצי md הועתקו בהצלחה.

פקודות לברק (Windows)

cd E:\bee-assets
git pull
pwsh research/scripts/sync-vault-and-graphify.ps1 -DryRun
pwsh research/scripts/sync-vault-and-graphify.ps1 -SkipGraphify
pwsh research/scripts/install-git-hooks.ps1
# באובסידיאן: חפש BRAIN / מוח

בדיקת mirror (WSL / Linux)

export BEE_VAULT_BEE_DIR=/tmp/bee-obsidian-mirror
bash research/scripts/sync-vault-and-graphify.sh --skip-pull --skip-graphify
